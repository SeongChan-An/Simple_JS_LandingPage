# JavaScript 연습을 위한 미니 프로젝트

드림코딩 엘리님의 미니 쇼핑몰 구현 연습

## 주요 기능

- JSON으로 구성된 Data로 List 생성
  - fetch와 async, await를 활용한 비동기 프로그래밍 연습
  - 리스트 영역에 overflow-y: auto; 를 통해 해당 영역 스크롤 생성
- 필터정렬을 위해 상단에 버튼을 통한 메뉴 구성
  - HTML 태그 속성을 이용하여 필터에 이용
